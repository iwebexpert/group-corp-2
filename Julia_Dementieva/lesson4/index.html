Самостоятельно разобраться с атрибутами тега script (async и defer)

Атрибуты async, defer тега script - асинхронные скрипты
решают проблему очевидной задержки постепенной загрузки html-файлов и скриптов браузером.

defer

Атрибут defer сообщает браузеру, что он должен продолжать обрабатывать страницу и загружать скрипт в фоновом режиме, а затем запустить этот скрипт, когда он загрузится.

  • Не блокируют страницу
  • Скрипты с defer всегда выполняются, когда дерево DOM готово, но до события DOMContentLoaded.
  • Отложенные с помощью defer скрипты сохраняют порядок относительно друг друга
  • Предназначен только для внешних скриптов

Используется для скриптов, которым требуется доступ ко всему DOM и/или важен их относительный порядок выполнения.

async

Скрипт абсолютно независим

  • Страница не ждёт асинхронных скриптов, содержимое обрабатывается и отображается
  • Событие DOMContentLoaded и асинхронные скрипты не ждут друг друга
  • Остальные скрипты не ждут async, и скрипты casync не ждут другие скрипты
  • Предназначен только для внешних скриптов

Используется для независимых скриптов, например счётчиков и рекламы, относительный порядок выполнения которых не играет роли.
